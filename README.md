# Oracle Next Education - Challenge Conversor de Moedas
Challenge Conversor de Moedas, parte do programa ONE da Oracle em parceria com a Alura

---

## Sobre o projeto
Esta aplicação de interação por texto, oferece um conversor de valores de moedas estrangeiras para o real brasileiro.

Foram utilizados no desenvolvimento:
- JetBrains IntelliJ IDE
- Biblioteca Google GSON para manipulação de JSON
- Java 21
- API ExchangeRate para obter os valores das moedas em relação ao real brasileiro

## Como funciona
- O app apresenta um menu com 10 opções de moedas para conversão
- Selecione uma das moedas (opções 1 a 10) ou digite 0 para sair da aplicação
- Ao escolher uma moeda, informe o valor que deseja converter
- A aplicação apresenta o valor convertido e a taxa de conversão
- Em caso de erro, você será informado e o app pedirá para digitar novamente
